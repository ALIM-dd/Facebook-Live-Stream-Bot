## Facebook Live Stream Bot

# Features:

· No Third Party APIs – fully self-contained
· Full Requests support for reliable performance
· Proxy integration required (supports HTTP/SOCKS)
· Multi-threaded streaming for stability
· Auto-reconnect on connection loss
· Schedule live streams in advance
· Custom titles, descriptions, and thumbnails
· Real-time viewer count & chat reader
· Logging system with error handling
